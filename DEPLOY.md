# ICCBI Alumni — Deployment Guide

## Stack
- **App**: Laravel 12 on [Render.com](https://render.com) (Web Service)
- **Database**: [Supabase](https://supabase.com) PostgreSQL (free tier)
- **Storage**: Render ephemeral disk (or swap to S3/R2 for persistence)
- **Push**: VAPID web push via `minishlink/web-push`

---

## 1. Supabase PostgreSQL Setup

1. Go to [supabase.com](https://supabase.com) → **New Project**
2. Choose a region close to your users (Singapore for PH)
3. Once provisioned, go to **Settings → Database → Connection string**
4. Copy the **URI** (format: `postgresql://postgres:<password>@db.<ref>.supabase.co:5432/postgres`)
5. Note these values for Render:
   - `DB_HOST` = `db.<ref>.supabase.co`
   - `DB_PASSWORD` = your Supabase DB password
   - `DB_USERNAME` = `postgres`
   - `DB_DATABASE` = `postgres`
   - `DB_PORT` = `5432`

### Enable SSL (required by Supabase)
Add to your production `.env`:
```
DB_SSLMODE=require
```
And in `config/database.php`, the `pgsql` driver already has `'sslmode' => env('DB_SSLMODE', 'prefer')` — this works.

---

## 2. Render Web Service Setup

1. Push this repo to GitHub/GitLab
2. On [render.com](https://render.com) → **New → Web Service** → connect your repo
3. Render will detect `render.yaml` automatically
4. Set the following **Secret** environment variables in the Render dashboard:
   - `DB_HOST` — from Supabase
   - `DB_PASSWORD` — from Supabase
   - `DB_USERNAME` — `postgres`
   - `VAPID_PUBLIC_KEY` — from your `.env` (generated locally)
   - `VAPID_PRIVATE_KEY` — from your `.env`
   - `MAIL_PASSWORD` — your SMTP password (if using email)

5. Deploy — Render runs `buildCommand` then `startCommand`

---

## 3. First Deploy Checklist

- [ ] `APP_KEY` is auto-generated by Render (set `generateValue: true` in render.yaml)
- [ ] `php artisan migrate --force` runs in buildCommand (creates all tables in Supabase)
- [ ] `php artisan storage:link` runs in buildCommand
- [ ] Create the first admin account via seeder or Tinker after deploy:
  ```bash
  # In Render Shell tab:
  php artisan tinker
  App\Models\User::create([
    'first_name' => 'Admin',
    'last_name'  => 'ICCBI',
    'email'      => 'admin@iccbi.edu.ph',
    'password'   => bcrypt('your-secure-password'),
    'role'       => 'admin',
    'status'     => 'approved',
    'education_level' => 'College',
  ]);
  ```

## 5. Storage Persistence (Important)

Render's ephemeral disk means uploaded files **are lost on redeploy**.
For production with persistent storage, switch to S3/R2:

```bash
composer require league/flysystem-aws-s3-v3
```

Then in `.env`:
```
FILESYSTEM_DISK=s3
AWS_ACCESS_KEY_ID=...
AWS_SECRET_ACCESS_KEY=...
AWS_DEFAULT_REGION=ap-southeast-1
AWS_BUCKET=iccbi-alumni
AWS_URL=https://your-cdn.com
```

Cloudflare R2 is free up to 10 GB/month and compatible with the S3 driver.
